# Goodmicro

The goodmicro repository contains various microcontroller software, mostly by
Tom Vajzovic.

This exists in Geo's private bitbucket account to store a branch of this open
source repository which also contains some source code written in work time or
for work projects.

The only current such project is the
[Nucleo-CAD-Logger](projects/nucleo_l452re/uart_decoder).

Please keep the branch `master` of this repository in sync with the branch with
the same name upstream repository at
[https://github.com/tcv-git/goodmicro](https://github.com/tcv-git/goodmicro).

All geo specific work goes on the branch `geo`.
